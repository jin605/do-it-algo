StringTokenizer 사용
st -> hasMoreTokens() / countToken()
st.nextToken() 하면 단어하나씩 꺼내오기시작

for문 사용
String t = br.readLine().trim();
if (t.isEmpty()) { // 입력이 비어있다면 0 출력
    System.out.println(0);
} else {
    int count = 1;
    for (int i = 0; i < t.length(); i++) {
        if (t.charAt(i) == ' ') count++;
    }
    System.out.println(count);
}
